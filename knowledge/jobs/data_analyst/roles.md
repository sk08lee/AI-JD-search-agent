# 数据分析岗位需求知识库

## 岗位类型

### 实习 internship

数据分析实习岗位通常更关注数据分析基础能力和工具使用。常见要求包括：

- 熟悉 Excel、SQL 等基础工具。
- 了解至少一种数据分析工具（Python/R、Tableau、Power BI 等）。
- 能参与数据清洗、数据整理和基础分析工作。
- 能输出数据分析报告和可视化图表。
- 有统计学基础、数学建模或校园数据分析项目经历会加分。
- 具备良好的逻辑思维和沟通能力。

### 校招 campus

数据分析校招岗位通常要求候选人具备扎实的分析能力和项目经验。常见要求包括：

- 熟练使用 SQL 进行数据查询和分析。
- 精通至少一种数据分析语言（Python 优先）。
- 熟悉数据分析库（Pandas、NumPy、Matplotlib、Seaborn 等）。
- 了解统计学原理和常用分析方法。
- 有完整的数据分析项目经历，能独立完成分析任务。
- 数学、统计学、计算机、经济学等相关专业背景。

### 社招 experienced

数据分析社招岗位通常更看重业务理解和分析深度。常见要求包括：

- 有完整业务数据分析或数据科学项目经验。
- 深入理解业务指标体系和数据驱动决策流程。
- 具备复杂数据建模和预测分析能力。
- 熟悉大数据技术栈（Hadoop、Spark、Hive 等）。
- 能设计数据分析框架和数据产品。
- 能带领团队完成复杂分析任务并推动业务落地。

## 高频能力关键词

- SQL
- Python/R
- 数据清洗
- 统计分析
- 数据可视化
- 业务理解
- 指标体系
- 数据建模
- 预测分析
- A/B 测试

## 工具与技术栈

### 数据处理

- SQL、Python（Pandas、NumPy）、R
- Hadoop、Spark、Hive、Flink

### 数据可视化

- Tableau、Power BI、Looker
- Matplotlib、Seaborn、Plotly

### 数据建模

- 机器学习算法、统计模型
- 特征工程、模型评估

## 与项目的匹配点

- 能使用 Python 进行数据处理和分析。
- 能使用向量嵌入技术进行语义检索。
- 能设计和实现数据分析流程。
- 能输出结构化的分析报告。
- 能将技术能力转化为业务洞察。
