# 软件工程师岗位需求知识库

## 岗位类型

### 实习 internship

软件工程师实习岗位通常更关注学习能力、编程基础和项目实践。常见要求包括：

- 熟悉至少一种主流编程语言（Java、Python、Go、C++、JavaScript/TypeScript 等）。
- 了解数据结构与算法基础。
- 了解计算机网络、操作系统等基础知识。
- 能参与代码编写、单元测试、Bug 修复和技术文档编写。
- 有开源项目贡献、校园编程比赛或个人项目经历会加分。
- 对新技术有热情，具备快速学习能力。

### 校招 campus

软件工程师校招岗位通常要求候选人具备扎实的技术基础和项目经验。常见要求包括：

- 扎实的数据结构与算法功底。
- 熟悉至少一种编程语言和相关技术栈。
- 了解数据库原理、分布式系统、网络协议等。
- 有完整的项目开发经历，能独立承担模块开发。
- 具备良好的代码规范和工程意识。
- 计算机、软件工程、电子信息等相关专业背景。

### 社招 experienced

软件工程师社招岗位通常更看重技术深度和业务经验。常见要求包括：

- 有完整项目从 0 到 1 或从 1 到 N 的开发经验。
- 深入理解系统架构、设计模式和技术选型。
- 具备性能优化、稳定性保障和故障排查能力。
- 熟悉至少一个技术领域（后端、前端、移动端、云原生等）。
- 能带领团队完成技术攻关和项目交付。
- 具备良好的技术文档和团队协作能力。

## 高频能力关键词

- 数据结构与算法
- 系统设计
- 数据库
- 网络协议
- 编程语言
- 框架与工具
- 代码质量
- 性能优化
- 故障排查
- 团队协作

## 技术栈分类

### 后端开发

- Java/Spring Boot、Go/Gin、Python/Django/Flask
- MySQL、PostgreSQL、Redis、MongoDB
- 分布式系统、微服务架构、消息队列

### 前端开发

- JavaScript/TypeScript、React、Vue、Angular
- HTML/CSS、Webpack/Vite、Node.js
- 性能优化、浏览器原理、前端工程化

### 移动端开发

- iOS/Swift、Android/Kotlin、React Native、Flutter
- 移动性能优化、跨平台开发、App 架构

### 云原生/运维开发

- Docker、Kubernetes、DevOps
- CI/CD、监控告警、自动化部署

## 与项目的匹配点

- 能使用 TypeScript/Node.js 构建后端服务。
- 能设计和实现 RESTful API。
- 能使用 MCP 协议集成外部工具。
- 能实现向量存储和检索功能。
- 能编写技术文档和单元测试。
